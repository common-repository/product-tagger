== Description ==

With this Plugin you can show all your wanted Products in the Sidebar with a Widget. 

Manual:

Multi-Products
To define the right Product use the Product-Tag like [woo_products_by_tags tags="shirt,food"]

Single-Product
To define the right Product use the Product-Tag like [woo_products_by_tags tags="food"]

When User click to the product in the Sidebar, they will forward to the product site.


== Installation ==

  1. Upload des Plugins in den Plugin-Ordner /wp-content/plugins/
